Introduction

Vuelidate Error Extractor

Vuelidate-error-extractor eases the pain of showing vuelidate error messages in your Vue components. Highly configurable, it can be tailored to your needs.

By registering a form-group component, vuelidate-error-extractor automatically shows all the errors for each input. The plugin can be configured to work with Foundation 6 and Bootstrap 3, as well as custom templates provided by the user.

The plugin is also able to work as a standalone component or in tandem with vue-i18n.

See the Pen Vuelidate Error extractor Usage Example by Dobromir (@dobromir) on CodePen.
